# Naira Refer & Earn Bot

Premium Telegram Mini App for Refer & Earn with Tasks and Bank Withdrawals (Nigeria 🇳🇬)

## Features
- Animated UI with confetti & bouncy effects
- Referral system with startapp param
- Task completion
- Wallet & Withdrawals (Flutterwave + Manual)
- Full Admin Panel
- Configurable via Admin

Built with React + Vite + TypeScript + Tailwind + Framer Motion

## Setup
1. `npm install`
2. Copy `.env.example` to `.env`
3. `npm run dev`
